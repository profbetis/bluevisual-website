The Hilbert Curve is a space-filling fractal curve. It has many computer science applications and I’ve run across its algorithm in the programs I use before. I took an interest to it and created it in various 2D forms until I wanted to try doing it in 3D. Houdini’s loopable node-graph system made it very simple and natural to set up.
